# 🐱 Kedi FotoKapan

Webcam + Hareket Algılama + Gemini AI + Telegram + Uzak Kontrol

## Uzaktan Komut Gönderme

`commands.json` dosyasını düzenle veya `uzak_kontrol.py` GUI'sini kullan.

### Desteklenen Komutlar
| Komut | Açıklama |
|-------|----------|
| `sistem_baslat` | Sistemi başlatır |
| `sistem_durdur` | Sistemi durdurur |
| `ekran_goruntu` | Ekran görüntüsü → Telegram |
| `mama_rapor` | Mama/su raporu → Telegram |
| `tam_test` | Tüm testler → Telegram |
| `kamera_test` | FFmpeg kamera testi |
| `gemini_test` | Gemini AI testi |
| `telegram_test` | Telegram bağlantı testi |
| `gece_modu` | Gece moduna al |
| `gunduz_modu` | Gündüz moduna al |
| `teamviewer_kapat` | TeamViewer kapat |
| `teamviewer_ac` | TeamViewer başlat |
| `teamviewer_yeniden` | TeamViewer yeniden başlat |
| `yeniden_baslat` | Programı yeniden başlat |
| `bilgisayar_yeniden` | Bilgisayarı yeniden başlat |
| `bilgisayar_kapat` | Bilgisayarı kapat |
